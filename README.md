<h3>Calculadora de Idade</h3>
<p>Neste projeto com conhecimentos iniciais em HTML, CSS e Javascript, o usuario digita um ano e é calculado uma idade a partir deste dado.</p>
<p>Competências: JavaScript · HTML · CSS</p>
